How to embed

The tool is an ES module; it imports its own dependencies (the shared orbit.js), so the page must be served over http(s) — ES modules do not load from file:// links.

In the markup, you need a DOMnode where you want the tether tool inserted:

<div id="insertItHere"></div>

Then import the function "tetherTool" in a module script and call it to initialise the tool:

<script type="module">
    import { tetherTool } from "./js/tetherTool_2.js";
    tetherTool(document.getElementById("insertItHere"))
</script>
